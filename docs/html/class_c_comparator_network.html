Detailed Description

A comparator network, each level of which is represented by a matching stored in an array m_nMatch. There is a comparator between channels j and k at level i iff m_nMatch[i][j] == k && m_nMatch[i][k] == j. This will allow for fast verification of whether a comparator network is a sorting network. Note that this is different from the implementation in CMatching, which is optimized for search rather than verification.
