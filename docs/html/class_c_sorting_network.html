Detailed Description

CSortingNetwork combines a comparator network with a binary Gray code generator to test whether the comparator network sorts based on the Zero-One Principle ,which says that a comparator network is a sorting network if and only if it sorts all inputs made up of only zeros and ones. See Knuth Volume 3 for the details. Using a Gray code generator instead of a standard binary string generator speeds up the test.
